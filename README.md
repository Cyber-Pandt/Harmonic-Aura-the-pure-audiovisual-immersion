# Harmonic-Aura-the-pure-audiovisual-immersion
A real-time 3D audio visualizer built with Three.js &amp; the Web Audio API . Upload any track to see it come to life across four unique cinematic modes (Aura, Rave, Quantum, Romance). Features reactive geometries, dynamic lighting, and a sleek, auto-hiding glass UI for pure audiovisual immersion.
